Programm which solve square equation.

You accept 3 value: there are coefficients our equation. If a = 0, b = 0,c = 0 we have infinity number of roots. If a = 0, b = 0 u have not roots. If a = 0 we have line equation and we use LineSolver(b,c). If c = 0, x1 = 0 and we need LineSolver(a,b). If a != 0 , b != 0 , c != 0 we should calculate discriminant and compare with 0: we will have 0 roots if discriminant < 0, we will have 1 roots if discr. = 0, we will have 2 roots if discr. > 0
